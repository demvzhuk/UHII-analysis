# UHII-analysis
Repository containing Google Earth Engine (GEE) scripts for analyzing Urban Heat Island (UHI) trends in global cities (2003–2020). Includes code for data processing from MODIS Aqua, trend calculation, visualization, and CSV exports for course work.

Link for GEE: https://code.earthengine.google.com/f88af8b6d688b1ec52e6faab9b228e43
